Hi, In this article i like to share my quick review about WrapBootstrap, a marketplace for buying and selling themes/templates made using Twitter Bootstrap. Twitter bootstrap created big rush among web geeks because of its endless features. Using Bootstrap you can design professional looking site within some minutes. Now twitter bootstrap is one of the top 10 projects in Github. If you are planning for a  new website, then you should go with Bootstrap. It also got lots of support from design community. There are lots of sites available which helps you to take your Bootstrap experience to next level. Some of the notable sites are WrapBootstrap, Bootwatch, Build with Bootstrap and much more. Off those WrapBootstrap is a nice site which allow you to buy and sell Themes made using Twitter Bootstrap. In this article i am going to share my views about WrapBootstrap. Ok, Lets start.

WrapBootstrap Review

Initially i was in dilemma whether to use WrapBootstrap or not. Even i don’t find any review about this site. So i decided to give a shot and i bought my first theme. I made payment using Paypal and within some minutes download got started. Everything went great. I downloaded my first Theme. The theme is really great. No bugs, fully responsive, compatible with all browsers, etc., After that i bought some more themes, everything went great. So, WrapBootstarp is not a SCAM. You can buy themes from WrapBootstrap without the fear of getting cheated.

As a Web Designer and Bootstrap Expert, i decided to sell one of my theme. I uploaded my theme to WrapBootstrap and waited 2 weeks to get my theme approved. It seems that they release themes every two weeks as batches. Next day i got a email saying “You sold an item!”. I am very exited. I submitted two more themes and i got some sales. On 16th i received my payment via Paypal. Again, WrapBootstarp is not a SCAM. If you are a designers and like to earn some extra bucks, then you can sell your bootstrap themes without any fear.

Downside:

1. Basic features: Site lacks some of the basic features like categories, search, changing password. Update: Recently Admin announced that he is working on new features.

2. Another problem is Support. It takes more than 3-4 days to get reply.  Update: There are some improvements in support. I hope it will improve more future.

WrapBootstrap Review

Pros:
1. Huge number of themes.
2. Sorting themes based on price, rating, etc.,
3. Instant download after payment.
4. Paypal, Credit card and Debit card accepted.
5. Simple and easy to use UI.
6. Customers can rate themes.
7. Customers can contact designers/developers directly.
8. Live preview.

Cons:
1. Support need to be improved.
2. Site lack some basic features like Search, Categories, etc.,
3. Some low quality themes.

Recommended? : I highly recommend WrapBootstrap if you are searching for decent Bootstrap Themes.

Overall Rating:
1. Quality of themes : 8/10.
2. Support : 7/10.
3. Payment : 10/10.
4. Site features : 7/10.

What they need to improve?
1. They need to improve support. They can have Support forum, so that people can help one another.
2. They need to maintain quality of themes by rejecting low quality themes.
3. They need to add more features to their site.

Twitter Bootstrap is the future of Web Designing. So, WrapBootstrap got great future. If they avoid about Cons, then WrapBootstrap will be a great place for Bootstrapers.
